MIDIMKtl - an MKtl class for MIDI.


One can find potential MIDI sources like this: 

MIDIMKtl.find;


// a posted line such as this

MIDIMKtl('???', -616253900);  // this is a nanoKONTROL


// give the one of interest a name, and make with the USB port ID:

MIDIMKtl(\nk1, -616253900); // lower USB port on my MBP

// A Test, there is no uid like that, so this fails

MIDIMKtl(\nk1, 12345); // 


// lookup existing ones by name:

MIDIMKtl(\nk1); // look up again;


// it loads descriptions of all its elements from files: 

MIDIMKtl(\nk1).devSpecs;

// which can be posted conveniently

MIDIMKtl(\nk1).postSpecs;


MIDIMKtl(\nk1).elNames.postcs; "";


(

MIDIMKtl(\nk1).addFunc(\sl1_1, \yubel, { |who, what, howmuch| 

"YAYAYAY: ".post; [who, what, howmuch].postln;

});

) // 

MIDIMKtl(\nk1).removeFunc(\sl1_1, \post);
